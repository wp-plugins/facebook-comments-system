== Description ==

Facebook Comments System allows you to easily add facebook comments to your wordpress website.

Facebook comments allow users to comment on entry using their facebook account.

Some features:

*   Nice option page layout
*   Option where to display facebook comments (posts, pages, both)
*   Change color scheme (dark/light)
*   Change width (responsive included)
*   Change other facebook comments api settings (order by, number of posts, mobile, html5)
*   Premium support
*   And more..

== Installation ==

1. Upload `/facebook-comments-system/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Change settings and have fun.
